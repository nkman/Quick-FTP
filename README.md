Quick-FTP
=========


Quick-FTP is a software for Quick file transfer over a TCP/IP network.



Version
----

1.0

How to use
--------------

```sh
git clone git@github.com:nkman/Quick-FTP.git
cd Quick-FTP
sudo make

For Server:
    run ./server

For Client
    run ./client
    
```

Supported Commands
-----------

```sh
ls - directory listing.
cd <path> - change directory.
get <file> - get file.
```
